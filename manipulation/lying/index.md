### a liar is born

	- waits when no one's around to do nasty stuff
	- fake cries to get attention
	- pretend to be deaf to keep playing
	- hiding forbidden stuff
	- hiding to do forbudden stuff
	- pretends to be hurt when others get hurt
	- "i don't want it anyway"
	- "she did it, not me"

lying is hard work
	- requires clear memory
	- sense of believability
